# Spark2 Documentation Hub

This is a simple static webpage project for presenting Spark2 robotic arm and end-effector documentation and providing download access to related materials.

## Folder Structure

- index.html: homepage content
- styles.css: page styling
- downloads/: download file directory

## How to Use

1. Open index.html directly in a browser to preview the page.
2. For GitHub Pages deployment, publish the repository contents as a static site.
3. Add real documentation files to the downloads folder to expand the page into a fuller resource hub.
