# 机械臂及末端设备说明文档

本文档用于说明机械臂及其末端设备的整体结构、接口特性、软件能力与可交付资料内容，旨在为系统集成、调试验证和后续扩展提供统一参考。

## 1. 机械说明

### 1.1 法兰接口

机械臂末端配备标准法兰接口，用于安装夹爪、相机支架以及其他末端执行器。法兰接口应包含清晰的安装尺寸说明，包括孔位、孔径、螺纹规格、定位孔尺寸和坐标系定义，以保证安装准确、重复性高且结构稳定。接口还需具备足够的刚度与承载能力，并与机械臂末端坐标系保持一致，或提供明确的固定变换关系。

### 1.2 夹爪安装说明

机械臂末端可通过法兰接口完成夹爪安装，以满足基础抓取任务。夹爪安装方案应涵盖安装接口尺寸、转接件设计、相对位姿定义以及关键参数说明，例如开合范围、夹持力、重量、通信方式和控制接口。若夹爪需要独立驱动或 SDK，则应同步提供相应说明，便于系统集成。

### 1.3 相机安装说明

系统建议在机械臂末端或机体上安装 Intel RealSense D435 深度相机。安装支架需保证定位准确、连接稳固，并尽量避免影响夹爪工作空间与机械臂运动范围。安装完成后，应提供相机坐标系与机械臂末端或基座坐标系之间的标定方法，便于视觉与运动控制协同。

## 2. 软件说明

### 2.1 通信连接说明

机械臂与上位机或控制器之间建议采用有线通信方式，常见方案包括 Ethernet / IP、CAN / CAN-FD、USB 以及 USB 转 CAN / 串口等。说明中应包含支持的通信接口、线缆规格、接口定义、连接方式以及配置参数，确保通信链路稳定可靠。

### 2.2 SDK 说明

SDK 是系统集成的基础接口层，优先推荐 C++ 版本；若暂时无法提供 C++ SDK，则可提供 Python SDK。SDK 应包含安装说明、依赖说明、基础示例、通信初始化、状态读取、关节控制和安全退出等能力，便于在 Linux 环境下接入 ROS、ROS2 或其他控制框架。

### 2.3 初始化与状态接口

SDK 应提供通信建立、握手确认、初始化流程、状态查询和异常反馈等接口。系统能够判断当前连接状态、控制状态以及是否处于可控状态，并在通信异常时给出明确的错误信息。

### 2.4 关节状态与控制接口

SDK 应支持读取关节位置、速度和力矩反馈，并提供关节控制接口。控制接口需要满足较高的控制频率，支持连续发送关节期望位置，并具备限位保护与安全停机机制。

### 2.5 MIT 控制模式说明

在需要高带宽运动控制时，机械臂可提供 MIT 控制模式。该模式以关节位置、速度、增益和前馈力矩为输入，通过反馈控制实现高频率运动跟踪。说明中应包含控制公式、参数范围、限幅规则以及进入与退出控制模式的流程。

### 2.6 停止与安全控制

系统应提供 Disable、Damping、自锁、急停等安全控制手段，以便在异常情况或测试阶段快速中断控制并保证设备处于安全状态。

## 3. 描述文件与文档说明

### 3.1 URDF 描述文件

应提供标定后的 URDF 文件，用于描述机械臂的关节结构、连杆关系、坐标系定义、限位参数、碰撞与视觉几何以及惯量信息。若系统安装夹爪或相机，应提供相应的扩展模型或固定变换说明。

### 3.2 SDK 使用说明

SDK 使用说明应包含安装方法、依赖环境、通信配置、初始化流程、状态读取示例、关节控制示例、MIT 控制示例、错误码说明和常见问题排查方法。

### 3.3 上位机使用说明

上位机说明应包括安装方式、连接方法、版本查看、状态查看、手动控制、模式切换、使能与失能、急停和故障恢复流程。

## 4. 交付物与优先级说明

后续可根据实际项目进展，逐步补充并发布相关资料文件，包括法兰图纸、夹爪与相机安装方案、URDF 文件、SDK 文档、示例代码以及上位机说明。优先级建议按基础可集成、可验证与可维护三个维度进行推进。
